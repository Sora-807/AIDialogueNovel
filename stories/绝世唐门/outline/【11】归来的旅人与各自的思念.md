# 第十二章

## 归来

1. 交流团返程——霍雨浩在路上沉默了很多
2. 橘子的笑容和那句"下次再来"一直在脑海里挥之不去
3. 到达史莱克城门时已是夜晚——宿舍楼下站着一个粉蓝色的身影
4. 王冬（双手抱胸）："去了这么久，还以为你死在外面了。"
   霍雨浩："……你等了多久？"
   王冬："谁等了？只是睡不着出来走走。"
   但她的发梢上沾着夜露——显然走了不止一会儿。

## 情报汇报

1. 霍雨浩向海神阁汇报日月帝国的情报——魂导技术、军力部署、政治格局
2. 穆老听完后沉默了很久，然后说："时间不多了。"
3. 唐三在神界注视着这一切——他的布局需要加速了

## 七怪集结

1. 贝贝在唐门旧址召集七人——"从今天起，你们就是新一代史莱克七怪。"
2. 霍雨浩、王冬、贝贝、徐三石、萧萧、江楠楠、和菜头——七人站成一排
3. 贝贝难得没有笑："这个名字不是荣誉，是责任。从这一刻起，我们的命绑在一起。"
4. 王冬看着霍雨浩的侧脸。他在想什么，她不知道。但她决定——无论他想的是什么，她都会站在旁边。
