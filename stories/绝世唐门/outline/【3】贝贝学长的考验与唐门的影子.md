# 第四章

## 贝贝的"偶遇"

1. 食堂里——徐三石和贝贝正在抬杠，霍雨浩和王冬端着餐盘路过
2. 贝贝用"友好的切磋"测试霍雨浩——蓝电霸王龙的雷光逼出灵眸的极限
3. 霍雨浩在压力下爆发精神冲击——贝贝被正面击中
4. 贝贝（笑）："有意思。雨浩，有没有兴趣加入一个……嗯，社团？"
5. 王冬插嘴："什么社团？"
   贝贝："唐门。虽然现在破落了点。但我们会重建的。"

## 唐门现状

1. 贝贝带两人参观唐门旧址——曾经的大陆第一宗门如今只剩一栋旧楼
2. 唐雅出现——唐门末代门主，一个独自支撑着破败宗门的女孩
3. 唐雅："贝贝，你又拐人。"
   贝贝："正经招募，不叫拐。"
4. 霍雨浩看着唐门的历史陈列，沉默了很久——他在这座破败的宗门里看到了某种和他自己相似的东西
