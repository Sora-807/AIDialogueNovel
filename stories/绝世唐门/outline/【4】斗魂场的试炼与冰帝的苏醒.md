# 第五章

## 斗魂场

1. 新生第一次独立出战——霍雨浩的对手是一名擅长速度的敏攻系学员
2. 战斗胶着——灵眸的精神能力被速度压制
3. 危急时刻，霍雨浩体内另一股力量爆发——极寒之气从掌心涌出
4. 冰帝的声音在他脑海中响起："终于醒了？人类。"
5. 霍雨浩以冰爆术瞬间冻结对手——全场震惊
6. 第二武魂「冰碧帝皇蝎」首次展露

## 徐三石与江楠楠

1. 徐三石在现场——本来是来看贝贝笑话的，结果看到了江楠楠
2. 江楠楠正在场边做记录（她担任斗魂场的见习裁判）
3. 徐三石："贝贝——那个女孩是谁？"
   贝贝："江楠楠。别想了，她不搭理人。"
   徐三石："巧了，我最擅长被人不搭理。"
4. 徐三石第一次正式搭讪——以江楠楠面无表情地转身离开告终
