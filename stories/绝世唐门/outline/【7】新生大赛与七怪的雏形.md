# 第八章

## 新生大赛

1. 全年级新生分组对战——最终胜出的队伍将获得进入内院考核的推荐资格
2. 霍雨浩+王冬+萧萧组成三人小队
3. 萧萧的九凤来仪箫在后方提供增幅和治疗
4. 三轮比赛——每一轮对手都更强，但三人的配合越来越流畅
5. 半决赛：霍雨浩第一次在公众面前同时使用精神探测+冰爆术
6. 决赛：对手是三人的两倍人数——王冬主动提出用那招

## 黄金之路

1. 霍雨浩："你确定？我们还没完全练成——"
   王冬："啰嗦。到时候跟着我就行。"
2. 灵眸的精神之力与光明女神蝶的光之力在空中交汇
3. 「璀璨中的凋零，黄金之路」——不完全版，但足够震撼全场
4. 贝贝在看台上站起来——不是惊讶，是若有所思
5. 赛后的夜晚——贝贝在唐门旧址对唐雅说："今年的新生，有几个不错的"
