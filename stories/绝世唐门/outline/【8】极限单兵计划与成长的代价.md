# 第九章

## 极限单兵计划

1. 海神阁的宿老会秘密选定了霍雨浩——他的双生武魂+精神力+魂导技术是最完美的候选人
2. 穆老（海神阁阁主）亲自指导霍雨浩——老人对这个沉默而坚韧的孩子有特别的关注
3. 训练内容包括：极限体能、精神力控制、冰属性精确操控、魂导器实战应用
4. 每天训练结束后霍雨浩几乎无法走路

## 王冬的沉默

1. 霍雨浩深夜回到宿舍——王冬还没睡
2. 王冬什么都没问。她把一杯温水放在霍雨浩床边
3. 霍雨浩："你还没睡？"
   王冬："睡不着。和你没关系。"
4. 霍雨浩累得说不出话，闭上眼睛就睡着了
5. 王冬坐在自己床上——听着他的呼吸声慢慢变均匀，才关灯
