# 魂师等级体系

斗罗大陆的魂师按修为分为以下等级：

| 等级 | 称号 | 魂环数 | 说明 |
|------|------|--------|------|
| 1-10级 | 魂士 | 0 | 未获取魂环，尚未成为正式魂师 |
| 11-20级 | 魂师 | 1 | 获取第一个魂环后成为魂师 |
| 21-30级 | 大魂师 | 2 | |
| 31-40级 | 魂尊 | 3 | |
| 41-50级 | 魂宗 | 4 | |
| 51-60级 | 魂王 | 5 | |
| 61-70级 | 魂帝 | 6 | |
| 71-80级 | 魂圣 | 7 | 拥有武魂真身，实力质的飞跃 |
| 81-90级 | 魂斗罗 | 8 | |
| 91-95级 | 封号斗罗 | 9 | 大陆顶级强者，可获封号 |
| 96-98级 | 超级斗罗 | 9 | 封号斗罗中的超级强者 |
| 99级 | 极限斗罗 | 9 | 无限接近神的境界 |
| 100级+ | 神 | — | 已成神级，脱离凡人范畴 |

## 魂环年限

- 白色：十年魂环
- 黄色：百年魂环
- 紫色：千年魂环
- 黑色：万年魂环
- 红色：十万年魂环
- 金色：百万年魂环
